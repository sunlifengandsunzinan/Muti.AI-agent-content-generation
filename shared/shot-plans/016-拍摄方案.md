## 二、运镜设计

| 分镜段 | 运镜方式 | 操作说明 |
|--------|---------|---------|
| 0-3s 跑步侧前方 | **固定机位** | 手机放路边支架，拍一条跑过的侧前方镜头 |
| 3-5s 看手表 | **固定机位** | 手机低位拍近景，跑者自然低头看手表前先做"瞟一眼"动作，然后看→苦笑→摇头（一条拍完） |
| 5-7s 生活场景 | **手持自拍** | 前后摄切换自拍，或手机横放购物车上固定拍（追求随手感，不要稳得像三脚架） |
| 7-9s 跑步正面 | **固定机位** | 手机放路边正面拍跑者从远处跑来→渐慢→看向镜头 |

> **粗糙感加分说明：** 不需要精致，不需要一镜到底。跑步段固定机位拍1-2条跑过素材即可，超市段随手一掏手机拍。

## 三、机位设置

| 场景 | 设备 | 设置方式 | 备注 |
|------|------|---------|------|
| 跑步·侧前方 | 手机 | 支架放路边，离跑步路线约2米远 | 手机高度≈人腰部，微微仰拍跑步姿态 |
| 跑步·看手表 | 手机 | 支架放低位，离站立位置约1米 | 俯视角拍手表+面部反应 |
| 超市/加油站·生活场景 | 手机 | 前摄自拍或放购物车 | 手持不用太稳，要随手感 |
| 跑步·正面 | 手机 | 支架放正前方路面，离跑步路线约5米 | 手机高度≈膝盖以上，拍跑者由远及近 |

**⚠️ 关键提醒：** 先拍跑步素材（第一条和最后一条），再拍超市素材，最后拍看手表近景。因为跑步出汗后状态不同。

## 八、拍摄执行清单

### 需要拍的素材（3-4条片段，可分段拍）

1. **跑步侧前方**（一条1-2次的过镜）
   - [ ] 手机放支架侧前方，拍一条跑过去的画面（约5秒，多跑几秒好剪）
   - [ ] 注意表情：跑到手机前自然低头看手表，皱眉

2. **看手表近景**（站定拍，无需真跑）
   - [ ] 手机放低位，俯拍手表+面部
   - [ ] 动作：低头→看表→皱眉→苦笑→轻轻摇头
   - [ ] 多拍2-3条选表情最自然的

3. **超市/菜市场生活场景**（抽空拍，随手拍质感）
   - [ ] 前摄自拍或放购物车固定
   - [ ] 拍正在装袋/看价签/掏钱的日常动作（不用看镜头）
   - [ ] 拍5-10秒，后期取2秒
   - [ ] 顺手拍一个超市空镜（环境音备用）

4. **跑步正面收尾**（与第一条同一时段拍，省事）
   - [ ] 手机放正前方路面固定
   - [ ] 从远方跑过来→到手机前自然放慢→看向镜头（"你懂的"表情）
   - [ ] 最后这句需要口型同步，「跑不掉」三个字口型要对上

### 注意事项
- [ ] 跑步和收尾可以同一条路同一个时间段拍，省时间
- [ ] 看手表近景可以跑完后原地站定拍，不用再跑
- [ ] 超市镜头随时可补，不一定非要和跑步同一天
- [ ] 先检查手机内存和电量
- [ ] 穿平时跑步的衣服（不用特意换），保持真实感
